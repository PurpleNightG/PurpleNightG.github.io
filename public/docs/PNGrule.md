## 紫夜请假细则
为了防止大家进群后不说话，不出警，在群内挂机的现象发生，根据紫夜规定，请各位按需请假，并合理把握请假时间哦~

### 请假标准

1. 超过一个星期无法来参加新训的
2. 有明确请假理由的 **（任何理由都可以，我们需要进行记录）**
3. 还有继续玩的想法的
4. 无特殊情况请假时间不超过半年的

### 请假格式

在请假时，请 **@ 或私聊群内管理**，并根据要求**写明请假开始时间与结束时间**，说明**请假原因**，具体格式如下：

```text
请假时间，请假原因
```
例如：
```text
2024年1月1日-2024年6月1日，上班
```
无特殊情况下，我们都将给予**准假**回复。
### 其他说明

值得注意的是，我们**不允许连贯性的短期请假**，因此根据规定，我们只允许你请3次假（即无特殊情况下最长1.5年），且两次假间隔时间不少于2个星期。超过3次的都将以默认退队处理。

**所以请各位合理把握请假时间，待你们时间充裕，我们再欢迎你的回归！**

## 关于黑点

根据紫夜相关规定，我们对新人及紫夜队员的所有行为进行记录并保留档案。关于黑点部分作出以下说明：

在教官警告过的前提下，屡教不改、一意孤行的，犯一次记录一个黑点，一个月内再犯则累计，黑点数记满4个则踢出工会。未满4个的，活跃状态14天消除全部黑点。

## 关于新训预约

入群后查看群公告，在 **3 天之内**@教官（群内管理）预约新训，满第3天后由教官提醒并给出最后期限，若无故不回复并不参加新训者，将于最后期限踢出本群！（可以重新加入）

## 紫夜新训群管理规定（部分）

由于紫夜新训组只接待新训以及需要回炉重造的老队员，因此我需要对新训群的人员去留作出以下说明：

### 违反规定退队条件

1. 因为个人原因导致黑点次数满4个的，踢出公会。
2. 进群后3天之内未预约新训并未请假的，且在提醒后不回复的，踢出公会。
3. 超过一星期未参加新训并未请假的，且在提醒后不回复的，踢出公会。
4. 成为正式队员超过一个月未参加训练并未请假的，且在提醒后不回复的，踢出公会。

### 达到要求退队条件

这与踢出公会概念不同，这是指**已通过入队考核**的队员，在一个月后将被踢出新训组。根据规定，新训组只容纳新训队员，因此没有申请留队，并入队考核通过的队员，将被踢出新训组，加入紫夜正式成员群。

## 紫夜留队申请

此申请只能由通过入队考核的队员发起，并且有一定要求。

根据紫夜规定，正式队员不允许待在新训组，因此将在入队考核通过的1个月后踢出新训组，如想继续留在新训组，可向群内管理申请，并给出正当理由。

转正的队员，有以下情况之一，**可以留在新训群**：

1. 转正式队员一个月内
2. 在新训群期间，保持长期活跃，并且出警次数较多
3. 拥有一定的执教能力，会简单基础教学（鲨鱼审核）
4. 有过留新训群的申请许可

离开新训群后，以下方式可以申请进入新训群：

1. 直接向管理组申请加入新训群（需理由）
2. 有执教的能力，能够通过鲨鱼的监考
3. 重新学习紫夜sop的老队员或档期断掉重学的新队员
4. 来紫夜的新队员

## 紫夜新训考核细则

在所有新训课程结束后，只有教官认定你可以参与考核，即**“新训准考”**头衔，方可预约入队考核。在入队考核通过后，方可成为紫夜正式队员并进入紫夜公开交流群。
如需预约考核，请在群内直接艾特群管理 **【鲨鱼SharkPepper Prime】** 预约考核时间。如有搭档，请与搭档一同预约考核，如果是单独考核，可以找其他正式队员或者教官陪考，**但评判标准和过图难度也会因此改变。**
入队考核所选用的地图将随机抽取合适地图作为考核图。在开始考核之前，请确认你已经对你之前所学的所有战术都熟记于心，并能够熟练运用它们。
在预约考核之前，我需要对本次考核作以下说明：

1. 此考核一个月只能约一次，每次考核有三次机会，特殊情况除外。
2. 考核前需要注意不要装影响考核难度的mod，具体装备携带限制考官会在考核前提出。
3. 考核过程中，不允许击打人质，否则本次考核失败，若失去三次机会则算作考核不通过。
4. 考核是否通过并不取决于是否清理完匪徒，而是取决于你所做出的战术动作以及决策是否合理，且两人表现分开计算，无须担心拖累他人或被拖累。
5. 考核的最终解释权，由紫夜的考官和会长所有。

那么，各位紫夜的新训队员们，祝你们好运！

## 紫夜不允许多队籍成员

紫夜不支持多队籍队员进入本群，训练过程中可以一起参与但是不能给新兵教学。在训练中也不得打断教学，有问题可以教学结束后再进行讨论。

已有队籍或隐瞒队籍人员将踢出本群。
