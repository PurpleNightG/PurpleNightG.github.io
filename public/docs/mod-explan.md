## Gunfighter 6 系列

> Gunfighter 6 是一个综合性模组包，提供了多个子模组来增强游戏体验。

### 1. 基础功能包

**文件名**：`pakchunk99-Mods_Gunfighter_6_Basic_P`

**核心功能**：

1. **武器枪托缩短**
   - 武器靠近物体时可自动缩短枪托
   - 注：该功能目前可能失效

2. **后坐力控制优化**
   - 单发射击后武器会快速回正（速度比原版快数倍）
   - 支持原本不具备此功能的武器
   - 枪感更加流畅，但仍需根据距离控制射速
   - 注：作者保留合理后坐力，不会改成 COD 那样几乎没有后坐（.300BLK 亚音速弹除外）

3. **惯性拖拽减少**
   - 减少大部分武器在开火和移动时的惯性拖拽感

4. **装备槽位扩展**
   - 轻型、中型、重型护甲均提供 **25 个装备槽**

5. **持枪姿势模拟**
   - **主武器**：不开镜时低于视线，视野更开阔，同时轻微内倾（模拟现实低姿持枪）
   - **副武器**：不开镜时在视野中央，轻微内倾（模拟现实紧凑持枪）
   - 枪口不会真正向下偏移，避免操作不便

6. **破门霰弹枪优化**
   - 靠近门锁时，持枪角度变化更加真实

7. **护甲配件**
   - 轻型护甲也可以装备侧甲（原本只有重型护甲可以）

8. **开镜速度提升**（制作中）
   - ADS（瞄准）速度将会提升

### 3. 移动优化包

**文件名**：`pakchunk99-Mods_Gunfighter_6_Mov_P`

**主要改进**：

1. **移速提升**
   - 全局移速提升至原版的 **104.7%**（相比 V5 版本提升 0.1%）

2. **受击惩罚系统**
   - **手臂受伤**：双臂重伤后主武器难以控制，即使开镜也会增加后坐力，建议降低射速或切换副武器
   - **腿部受伤**：
     - 每条腿受伤减少 6% 移速
     - 双腿重伤（红色状态）共减少 12% 移速
     - 双腿折断最多减少 24% 移速
   - 注：后续将继续优化伤害惩罚系统，不会采用一击即死设定

3. **自由视角扩展**
   - 视角角度提升至 **120 度**（左右各 60 度）

4. **侧身移速提升**
   - 侧身移动时移速提升 **25%**

5. **视觉效果优化**
   - 减少受击和被压制时的画面模糊
   - 模糊效果恢复速度更快

6. **翻越高度优化**
   - 轻微提高可翻越的台阶高度
   - 更容易跨过障碍物，适配部分模组地图

### 4. 弹道优化包

**文件名**：`pakchunk99-Mods_Gunfighter_6_Ballistic_P`

**弹药系统改进**：

#### 4.1 通用弹药调整

- 提高所有弹药的命中率和肢体伤害
- 大幅增强停止力（敌人中枪后会有明显反应，不会像"钢铁侠"那样硬扛）
- 所有弹药（尤其是 JHP 弹）具有一定肢解效果（需多发触发）

#### 4.2 移动射击优化

- 移除移动时的额外减速效果
- 减少钢板造成的跳弹惩罚

#### 4.3 .300BLK 弹重做

- **弹药类型**：视为亚音速弹
  - AP 弹重命名为 **FMJ**（AAC 200gr）
  - JHP 弹重命名为 **OTM**（Fed 220gr）
- **距离衰减**：150-200 米外伤害大幅衰减
- **穿透力调整**：
  - JHP/OTM 无法打穿 3A 级护甲（如《神父之罪》《霓虹坟墓》中的敌人）
  - AP/FMJ 可以打穿但需要多发命中
- **近距伤害**：对无防护目标伤害轻微提升

#### 4.4 5.56mm / 7.62mm 弹药调整

- 轻微优化伤害曲线，100-200 米外表现更好（游戏内效果不明显）
- 5.56 AP 弹穿透力提升，更容易打穿敌人防护

#### 4.5 9mm 弹药调整

- 轻微优化伤害曲线，30-50 米外表现更好

#### 4.6 破障弹重做

- 大幅提升对躯干的伤害
- 一击即可制服敌人

## InGameMenu 系列

> 提供游戏内菜单和蓝图加载功能的模组系列。

### 1. 简易模组菜单

**文件名**：`pakchunk99-Mod_SimpleModMenu`

**特点**：
- ✅ 独立运行，无需依赖其他模组
- 📱 打开方式：
  - 按键盘 **M** 键
  - 或点击主菜单新增的 **SMM** 按钮

<div class="note info">
如果有地图 DLC 但对应标签未激活，启动一次快速单人游戏即可完成验证。
</div>

**配置文件路径**：
```
%localappdata%\ReadyOrNot\Saved\SaveGames\SMM\
```

### 2. 简易蓝图加载器

**文件名**：`pakchunk99-Mod_SimpleBlueprintLoader`

**功能说明**：

## 重生模组

**文件名**：`pakchunk99-RespawnPlayer_P`

<div class="note warning">
该模组可以不配置使用默认设置，如需自定义重生次数请按以下步骤操作。
</div>

### 配置文件操作步骤

1. **找到配置文件**
   
   文件路径：`\AppData\Local\ReadyOrNot\Saved\Config\Windows\Engine.ini`

2. **快速打开方法**
   
   在文件资源管理器的地址栏输入 `%appdata%`，按回车进入 AppData 文件夹，再按照上述路径找到 `Engine.ini` 文件

3. **添加配置**
   
   将下方的配置代码复制粘贴到 `Engine.ini` 文件的**最底部**

### 注意事项

<div class="note info">
<b>为什么复活次数为 0 时会切换到自由视角？</b><br>
这是因为游戏本身的代码存在问题 —— 如果玩家复活后再次阵亡，观战功能会失效。因此该模组采用"复活次数为 0 时自动切换自由视角"的方式来规避这个 bug。
</div>

### 配置参数说明

```ini
[/Game/Mods/_SBLdata/BP_RespawnPlayer.BP_RespawnPlayer_C]
; 玩家复活所需时间（每个玩家独立倒计时）
; 单人模式下，这个延迟时间为 0 秒
RespawnDelay=10

; 玩家团队的最大复活次数
; 当复活次数为 0 时，阵亡玩家会自动切换到自由视角
MaxSpawnNum=3
```
